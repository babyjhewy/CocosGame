# Coco's Jewel Garden

A cosy mobile match-3 makeover game for Coco, built for GitHub Pages.

## Play

GitHub Pages URL:
https://babyjhewy.github.io/CocosGame/

## Included

- Warm cafe home screen
- Coco lobby
- Level map
- Playable match-3 board
- Goals, moves, score and power-ups
- Pause menu
- Daily bonus, events, inbox, friends and shop screens
- Room makeover screen
- Local save with localStorage
- Sound and vibration toggles
- Offline cache service worker

## iPad setup for Mum

Open the GitHub Pages link in Safari, tap Share, then tap Add to Home Screen.
